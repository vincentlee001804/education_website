Project Title: Education Website Project
Project Overview
You are required to design and develop a responsive education website that supports
undergraduate students in their learning journey. The website should combine learning
resources with a bookstore module for purchasing academic books. This project will test your
ability to integrate front-end, back-end, and database technologies while ensuring usability,
accessibility, and security.

Project Requirements
Your website must include the following components:
1. Homepage
o Clear navigation and engaging design.
o Responsive layout for desktop and mobile.
2. Course Content Pages
o Lecture notes, video tutorials, or interactive quizzes.
o Organized by subject or module.
3. Student Portal
o Login and registration functionality.
o Personalized features (progress tracking, saved resources).
4. Discussion Forum / Q&A Section
o Peer-to-peer learning and interaction.
o Moderation tools for managing posts.
5. Bookstore Module
o Book catalogue with categories and search functionality.
o Purchasing cart and checkout process.
o Database integration for inventory, user accounts, and purchase history.

6. Technical Requirements
o Front-end: HTML, CSS, JavaScript.
o Back-end: PHP, Node.js, or Python Flask.
o Database: MySQL, MongoDB, or equivalent.
7. Documentation
o Design choices and rationale.
o Implementation process.
o Reflection on usability, accessibility, and security.
